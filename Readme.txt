[1] How to compile and execute?
  Download this project to your Ubuntu machine, 'cd' into this directory and
	sh ./build.sh
  If everything is normal, you should see a GUI window appears. 

[2] What to do next?
  Press button 'Start(A)' and see the cars run in circles and try to cross the 
bridge occasionally and press button 'Stop'. Then try another button 'Start(B)' 
to see part (b) for another situation.
